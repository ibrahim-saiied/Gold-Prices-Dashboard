## Case Study
Gold Prices Dashboard is a daily auto-updating dashboard designed to track and analyze gold prices in Egypt.
Gold prices fluctuate every day, making it challenging for buyers to make informed decisions. This project provides a simple, interactive, and automated solution to monitor buying and selling prices for different gold carats and understand trends.

- Gold Buying Price: Daily price to purchase gold
- Gold Selling Price: Daily price to sell gold
- Price Trend: Direction of price movement over time

## Process
1- **Data Sourcing:**
- Found a website providing historical gold prices for the past year in Egypt.
- Extracted the data and applied ETL (Extract, Transform, Load) to clean, structure, and prepare it for analysis.

2- **Data Pipeline:**
- Needed an automated solution to fetch daily prices and append to the dataset.
- Used Google Sheets + App Script as a lightweight database alternative to store and manage data automatically.

3- **Data Modeling & Measures & Visualization**
- Created calculations for daily price changes, trends, and averages.
- Built interactive cards showing buying/selling prices per gold carat.
- Used HTML within Power BI to add animations and hover effects.
- Added trend analysis charts to help understand price directions for better decision-making.
